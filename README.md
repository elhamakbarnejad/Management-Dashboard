# Product & User Management Dashboard

This project is a _Product and User Management Dashboard built with React and Vite_.  
It is designed as a modern, scalable, and component-based admin dashboard with a focus on clean architecture and user experience.

---

## Features

- Modern and responsive UI (Responsive Design)
- Component-based architecture with clean folder structure
- Theme management (Dark / Light) using Context API and LocalStorage
- Custom React hooks for reusable logic (e.g. page title management)
- Data visualization using Recharts
- Product and user management tables with actions (visibility toggle, remove)
- Built with Tailwind CSS
- Client-side routing using React Router

---

## Technologies Used

- React
- Vite
- JavaScript (ES6+)
- Tailwind CSS
- React Router (createBrowserRouter)
- Context API
- Recharts
- LocalStorage
- Custom Hooks

---

## Install dependencies:

- npm install

---

## Run the project in development mode:

- npm run dev

## Notes

- The project currently uses mock data and can be easily connected to a real API.
- Product and user management sections are implemented with reusable table components.
- The application structure is designed for scalability and maintainability.
